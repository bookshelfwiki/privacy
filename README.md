# Privacy Policies

This repository contains the public privacy policies for bookshelfwiki applications.

The policies are organized by platform, as different platforms rely on different underlying technologies and services.

## Structure

* `/android` – Privacy policy for the Android version of the app (Google Play)
* `/ios` – Privacy policy for the iOS version of the app (App Store)

Each policy reflects the actual behavior of the respective app version, including any platform-specific dependencies.


## Privacy Principles

* The apps do not collect, store, or share user data.
* All processing (e.g. document scanning) is performed on-device.
* Platform-specific services (such as system frameworks or bundled libraries) may have their own data handling policies.
