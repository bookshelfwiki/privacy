# Privacy Policies

This repository contains the public privacy policies for bookshelfwiki applications.

The policies are organized by platform, as different platforms rely on different underlying technologies and services.

## Structure

* `docs/privacy/android/index.html` – Privacy policy for the Android version of the app (Google Play)
* `docs/privacy/ios/index.html` – Privacy policy for the iOS version of the app (App Store)

Each policy reflects the actual behavior of the respective app version, including any platform-specific dependencies.
